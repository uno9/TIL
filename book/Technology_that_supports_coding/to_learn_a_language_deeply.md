# 言語を深く学ぶには
作成日時:2021/6/7

## 1.言語を深く学ぶ方法
1. 言語を比較する
  - 言語の違いを知る事で言語の特徴が見える
    - PHPにしかない特徴、Rubyにしかない特徴
    - どの言語にも共通する書き方
2. 言語の歴史を学ぶ
  - なぜどの言語が必要になったのか？
  - どんな課題があったのか？
3. 言語を自作する
  - いままで学んだ事を自分で開発に落とし込むことで、分からないことがさらに分かるようになる
