 # 💻 google_translate_api
コマンドからgoogle翻訳を行うアプリの使用方法

## 🚀 Usage
```bash
# 英語翻訳
$ node google_translate/translate.mjs こんにちは en
翻訳結果 > 
 Hello

# 日本語翻訳
$ node google_translate/translate.mjs Requirements ja
翻訳結果 > 
 要件
```
