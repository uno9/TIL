# svelte first app
作成日時:2021/2/24

## 1.install
* nodeのインストールを実行する
"../node/node_install.md"
* svelteでアプリを作成
`npx degit sveltejs/template svelte-project`
* svelte appの起動
```
cd svelte-project

npm install
npm run dev
```
